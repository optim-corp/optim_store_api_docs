Contract Public Key Endpoint

OPTiM Store のユーザー企業様が御社サービスを利用開始する際に行われる Contract Setup API の、リクエストの署名を検証する際に利用する公開鍵の公開APIです。

Request

Method

  • GET

Resource URL

  • URLは、OPTiM Store との契約時にご連絡いたします。

Header Parameters

Content-Type: application/json

Query Parameters

  • なし

Request Parameters

  • なし

Response

Response Code

200 OK

Response Parameters

key type description
keys array OPTiM Store が Contract Setup API の署名に利用する秘密鍵に対応する公開鍵(JWK形式)
